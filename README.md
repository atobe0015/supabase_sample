## 環境
```
Node.js v14.16.0 ~  
Next.js v12.1.5
```

### 初期環境構築

```
$ npm install

$ cp .env.local.sample .env.local
```

### 各種コマンド

```
環境立ち上げ
$ npm run dev

vitest実行
$ npm run vitest

lint実行
$ npm run lint

openAPIから型定義ファイルの抽出
$ npm run generate:aspida--RESAS
$ npm run generate:aspida--NextServer
```
